Como usar o book

Instrumento do consultor para o levantamento SAP B1. Cada book é de um cliente/projeto e salva sozinho na nuvem. Navegue por Cliente → Projeto → Book; dentro, há módulos → etapas → perguntas.

Consultor
  • Responda as perguntas do módulo. Em Sim/Não, o Sim abre os pontos de atenção (o que aprofundar na entrevista).
  • Toda pergunta deve ser classificada: Standard, GAP ou Fora de Escopo. A etapa fica verde só quando tudo está respondido e classificado.
  • Adicione, edite, remova e reordene perguntas, etapas e módulos conforme o cliente.
Gerente
  • Registre o aceite técnico: o módulo trava e o fluxograma é liberado.
  • Para editar depois de aprovado, gerente ou admin reabrem o módulo.
Admin
  • Editar modelo-mestre: muda o catálogo-padrão de que os books novos nascem (faixa vermelha indica que é o mestre).
  • Convide usuários por e-mail e defina o papel.
Todos
  • Relatório de GAPs: consolida a classificação e soma as horas — base para o comercial.
  • Exporte o módulo em HTML: versão interna (com classificação) ou cliente (sem).

Para rever este guia, clique em "Como usar" no topo da página.

Módulos do projeto

Marque os módulos que este cliente utiliza e use as setas para definir a ordem das abas. Só os marcados aparecem no levantamento. Desmarcar esconde o módulo, mas o conteúdo é guardado — se remarcar, ele volta como estava.

Relatório de GAPs e Fora de Escopo

Visão consolidada das classificações do levantamento. Os pontos GAP e Fora de Escopo são os que geram horas adicionais para o comercial negociar. O campo de horas é opcional.
